1. notes ===> 笔记文件夹存放处
2. src ===> 具体的文件
3. assets === > 静态资源
4. components === > 项目公共组件
5. hooks ===> 自定义hooks文件夹存放处
6. packages ===> 组件封装
7. router ===> 路由定义
8. layouts === > 存放布局页面
9. server || api 网络请求存放处
10. store ===> vue的状态管理文件
11. style ===> 公共样式文件
12. utils ===> 工具类存放处
13. views ===> 具体的前端页面存放处
